平台總覽

關於本平台

部北透析管理平台是專為透析室設計的完整管理系統，提供排班管理、護理分組、病人管理、 工作協作等功能，協助團隊提升工作效率與照護品質。

角色與權限

管理員

完整存取所有功能，包含使用者管理

編輯者

護理師角色，管理排班、分組、工作日誌

貢獻者

醫師角色，管理醫囑、檢驗報告、醫師班表

查看者

查閱權限，檢視日誌、耗材、醫師班表

側邊欄結構

共用功能區

  • 每日排程 - 當日透析排班
  • 護理分組 - 護理師分組分配
  • 我的今日病人 - 個人負責病人
  • 訊息中心 - 協作與通知

排班管理（管理員/編輯者）

  • 週排班 - 一週排班總覽
  • 床位總表 - 完整床位管理
  • 調班換床 - 例外排班管理
  • 預約變更 - 未來變更排程

後臺管理

  • 工作日誌、護理班表、KiDit 申報
  • 醫師班表、檢驗報告、藥囑管理
  • 每月耗材、統計報表、使用者管理
